Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
77 78 79

Подобный способ создания срезов берет элементы,
включая первый индекс, и до последнего индекса, не включая его.
```
